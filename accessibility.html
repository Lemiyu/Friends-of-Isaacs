1. Our commitment

Friends of Isaac's Food Bank CIC is committed to making this website accessible to as many people as possible, regardless of disability, assistive technology, or circumstance. We believe that everyone deserves equal access to information about our services — including those who may need help most.

This statement applies to the website at friendsofisaacs.co.uk (domain to be confirmed). It does not cover third-party websites or services we link to, such as GoFundMe, Facebook, or Amazon.

2. Standards we aim for

We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. These guidelines explain how to make web content more accessible to people with disabilities, including those with visual, auditory, physical, speech, cognitive, language, learning, and neurological disabilities.

WCAG 2.1 AA is the standard referenced by the Public Sector Bodies Accessibility Regulations 2018. Whilst we are not legally required to comply as a small CIC, we use it as our benchmark because it represents good practice.

3. Measures we have taken

In building this website, we have taken the following steps to support accessibility:

  • Semantic HTML — page structure uses appropriate heading levels, landmark regions (header, nav, main, footer), and meaningful element types throughout.
  • Keyboard navigation — all interactive elements including navigation, buttons, form fields, and links are reachable and operable using a keyboard alone.
  • Focus indicators — visible focus styles are present on all interactive elements so keyboard users can track their position on the page.
  • Colour contrast — text and interactive elements have been designed to meet or exceed the WCAG 2.1 AA minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text.
  • Alternative text — all meaningful images include descriptive alt attributes. Decorative images use empty alt attributes so they are skipped by screen readers.
  • Responsive design — the site is fully responsive and works at all viewport sizes, including when text is resized up to 200%.
  • Reduced motion — animations and transitions respect the prefers-reduced-motion media query and are disabled for users who have requested this in their system settings.
  • Form labels — all form inputs are associated with visible labels. Required fields are indicated both visually and programmatically.
  • ARIA attributes — ARIA roles and attributes are used where HTML alone is insufficient to convey state or purpose, such as on the mobile navigation toggle.
  • No content relying on colour alone — information is never conveyed by colour alone; shape, text, or icons are used alongside colour.

4. Known issues

We are not aware of any significant accessibility barriers on this website at the time of publication. However, as the site is actively developed and content is added, new issues may arise.

If you experience any difficulty accessing any part of this website, please let us know using the contact details in section 6 below. We take all feedback seriously and aim to resolve reported issues promptly.

5. Assistive technology compatibility

This website has been designed to be compatible with the following assistive technologies:

  • Screen readers including NVDA, JAWS, and VoiceOver (macOS and iOS).
  • Browser zoom up to 200% without loss of content or functionality.
  • Keyboard-only navigation.
  • High contrast display modes.
  • Voice recognition software including Dragon NaturallySpeaking.

The website is built using standard HTML, CSS, and JavaScript without reliance on frameworks that can introduce accessibility regressions. We do not use PDFs as the primary means of presenting information.

6. Feedback and contact

We welcome feedback on the accessibility of this website. If you experience any barriers, find content difficult to access, or need information in a different format, please get in touch:

  • Email: friendsofisaacs@hotmail.com
  • Phone: 07496 381323

We aim to respond to accessibility feedback within 5 working days.

7. Enforcement

If you are not satisfied with our response to an accessibility concern, you can contact the Equality Advisory and Support Service (EASS):

  • Website: equalityadvisoryservice.com
  • Phone: 0808 800 0082

This statement was prepared in January 2025. We will review and update it following any significant changes to the website.
